19 ## Usage
20 
21 **Initialization**
22 
23 Create a Serial Terminal object. This can be initialized with optional newline and delimiter characters.
24 
25 Default newline character: ```'\n'```
26 Default delimiter character: ```Space```
27 
28 ```c++
29 #include <SerialTerminal.h>
30 
31 // Newline character '\r' or '\n'
32 char newlineChar = '\n';
33 // Separator character between commands and arguments
34 char delimiterChar = ' ';
35 
36 // Create serial terminal object
37 SerialTerminal term(newlineChar, delimiterChar);
38 
39 
40 void setup()
41 {
42  // Initialize serial port
43  Serial.begin(115200);
44 
45  // Initialize the built-in LED
46  pinMode(LED_BUILTIN, OUTPUT);
47  digitalWrite(LED_BUILTIN, LOW);
48 }
49 ```
50 **Register new commands**
51 
52 Commands must be registered at startup with a corresponding ```callback handler``` . This registers the command only, excluding arguments.
53 
54 The callback handler will be called when the command has been received including the newline character.
55 
56 An example of registering multiple commands:
57 
58 ```c++
59 void setup()
60 {
61  ...
62 
63  // Add command callback handlers
64  term.addCommand("?", cmdHelp);
65  term.addCommand("help", cmdHelp);
66  term.addCommand("on", cmdLedOn);
67  term.addCommand("off", cmdLedOff);
68 }
69 
70 void cmdHelp()
71 {
72  // Print usage
73  Serial.println(F("Serial terminal usage:"));
74  Serial.println(F(" help or ? Print this usage"));
75  Serial.println(F(" on Turn LED on"));
76  Serial.println(F(" off Turn LED off"));
77 }
78 
79 void cmdLedOn()
80 {
81  // Turn LED on
82  Serial.println(F("LED on"));
83  digitalWrite(LED_BUILTIN, HIGH);
84 }
85 
86 void cmdLedOff()
87 {
88  // Turn LED off
89  Serial.println(F("LED off"));
90  digitalWrite(LED_BUILTIN, LOW);
91 }
92 ```
93 
94 **Set default handler**
95 
96 Optional: The default handler will be called when the command is not recognized.
97 
98 ```c++
99 void setup()
100 {
101  ...
102 
103  // Set default handler for unknown commands
104  term.setDefaultHandler(unknownCommand);
105 }
106 
107 void unknownCommand(const char *command)
108 {
109  // Print unknown command
110  Serial.print(F("Unknown command: "));
111  Serial.println(command);
112 }
113 ```
114 
115 **Read from serial port**
116 
117 Read from the serial port in the main loop:
118 
119 ```c++
120 void loop()
121 {
122  // Read from serial port and handle command callbacks
123  term.readSerial();
124 }
125 ```
126 
127 **Get next argument**
128 
129 Get pointer to next argument in serial receive buffer:
130 
131 ```c++
132 char *arg;
133 
134 // Get next argument
135 arg = term.getNext();
136 if (arg != NULL) {
137  Serial.print(F("Argument: "));
138  Serial.println(arg);
139 } else {
140  Serial.println(F("No argument"));
141 }
142 ```
143 
144 **Get remaining characters**
145 
146 Get pointer to remaining characters in serial receive buffer:
147 
148 ```c++
149 char *arg;
150 
151 // Get remaining characters
152 arg = term.getRemaining();
153 if (arg != NULL) {
154  Serial.print(F("Remaining: "));
155  Serial.println(arg);
156 }
157 ```
158 
159 **Clear buffer**
160 
161 Optional: The serial receive buffer can be cleared with the following call:
162 
163 ```c++
164 term.clearBuffer();
165 ```

167 ## Library configuration
168 
169 ```SerialTerminal.h``` contains the following configuration macro's:
170 
171 * ```ST_RX_BUFFER_SIZE``` : The default serial receive buffer size is 32 Bytes. This includes the command and arguments, excluding the ```'\0'``` character.
172 * ```ST_NUM_COMMAND_CHARS```: The default number of command characters is 8 Bytes, excluding the ```'\0'``` character.
